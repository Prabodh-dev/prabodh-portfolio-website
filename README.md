# Black Ghost Portfolio - Prabodh Raj

A modern, fully-featured portfolio website built with Next.js 15. Features a dark gaming aesthetic with an animated ghost mascot, lightning effects, and retro styling.

**⚠️ PRIVATE PROJECT - ALL RIGHTS RESERVED**

This is a personal portfolio website. The code, design, and content are proprietary and not licensed for reuse, modification, or distribution.

## ✨ Features

- **Animated Ghost Mascot**: Interactive character with smooth animations
- **Lightning Effects**: Dynamic visual effects
- **Portfolio Sections**: Hero, About, Skills, Projects, Certifications, Achievements, Experience
- **Downloadable Resume**: PDF download functionality
- **Contact Section**: Social links and contact information
- **Responsive Design**: Fully mobile-optimized with dark theme
- **Content Management**: Private admin system for content updates
- **Secure Backend**: Protected routes and authentication

## 🚀 Development Setup (Private Use Only)

### Prerequisites

## 🎮 Design Theme

**"Black Ghost" - Dark Gaming Aesthetic**

### Visual Identity
- **Color Palette**: Deep blacks, electric blues, neon accents
- **Typography**: Modern sans-serif with gaming influences
- **Animations**: Smooth transitions, ghost floating effects, lightning strikes
- **Mascot**: Animated ghost character that responds to interactions
- **Effects**: Dynamic lightning bolts, glowing borders, shadow depth
- **Layout**: Clean, modern cards with dark backgrounds
- **Mobile-First**: Fully responsive across all devices

### Key Components
- **Ghost Mascot**: SVG-based character with CSS animations
- **Lightning Effects**: Canvas-based dynamic lightning generation
- **Retro Elements**: Pixel-perfect borders and glowing effects
- **Dark Theme**: Eye-friendly dark color scheme throughout

## 🛠️ Tech Stack

- **Framework**: Next.js 15.5.12
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Authentication**: NextAuth.js

- **Frontend**: Server-side rendered React components
